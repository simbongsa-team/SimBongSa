# 심봉사 (心: 마음이 가는 봉사활동) - 최적 자원봉사 매칭 AI

## 💡 프로젝트 소개

**심봉사**는 대한민국 국민의 자원봉사 참여율을 높이기 위해 개발된 **AI 기반 최적 매칭 서비스**입니다.  
개인의 시간과 관심사에 맞는 자원봉사 활동을 추천하여 참여 장벽을 낮추고, 자발적 봉사 문화를 활성화하는 것을 목표로 합니다.

---

## 📝 제안 배경

### 문제 정의 및 현황

- 대한민국 자원봉사 참여율은 OECD 국가 평균보다 낮으며, 국민 10명 중 약 7~8명은 최근 1년간 봉사활동에 참여하지 않았습니다.

| 조사 기준 | 참여율 | 시사점 |
| --- | --- | --- |
| 사회조사 (최근 1년간) | 20%대 초중반 | 국민 대부분이 미참여 |
| 국제 비교 | OECD 평균 이하 | 시민 참여 수준 낮음 |


### 자원봉사 활동 미참여 원인

| 순위 | 미참여 이유 | 비율 | 문제의 핵심 |
| --- | --- | --- | --- |
| 1 | 시간적 여유가 없음 | 약 46% | 시간 부족 및 활동 과정 비효율성 |
| 2 | 관심 부족 | 약 32% | - |
| 3 | 참여 방법을 모름 | 약 5% | 정보 탐색 어려움, 진입 장벽 |
| - | 맞춤 활동 불명 | 기타 | 개인 특성을 고려하지 않는 획일적 정보 제공 |

**결론:**  
대부분의 잠재적 자원봉사자는 **시간 부족**, **참여 방법 미숙지**, **맞춤 활동 부족**으로 인해 봉사활동에 참여하지 못하고 있습니다.

---

## 🎯 서비스 목표

1. **시간 절약 & 효율적 매칭**  
   - AI 기반 추천으로 사용자가 직접 활동을 탐색하는 시간을 최소화
2. **개인 맞춤형 추천**  
   - 개인의 관심사, 시간, 활동 유형을 고려하여 최적 활동 추천
3. **지속적 참여 유도**  
   - 사용자가 꾸준히 활동할 수 있도록 경험과 피드백 기반 추천

---

## ⚙️ 주요 기능

- **회원가입 및 로그인**: JWT 기반 인증, React 프론트와 Node.js 백엔드 연동  
- **자원봉사 매칭**: LightFM 기반 추천 AI가 개인 맞춤 활동 추천  
- **활동 탐색**: 관심사, 시간, 지역 기반 필터링  
- **활동 기록 관리**: 참여 활동 기록 및 통계 제공
